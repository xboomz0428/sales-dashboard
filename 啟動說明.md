# 銷售數據分析系統 — 啟動說明

## 前置需求

請先安裝 Node.js（建議 v18 以上）：
https://nodejs.org/en/download/

## 啟動步驟

開啟命令提示字元（cmd）或 PowerShell，進入專案目錄：

```bash
cd C:\Users\User\sales-dashboard
npm install
npm run dev
```

然後用瀏覽器開啟： http://localhost:5173

## 使用方式

1. 將 Excel 檔案（.xls 或 .xlsx）拖放至上傳區
2. 系統自動解析並顯示 KPI 卡片與圖表
3. 左側篩選器可選擇年份、月份、通路、品牌
4. 頂部 Tab 切換：趨勢分析 / 通路分析 / 品牌分析 / 熱力圖

## 欄位對應

系統會自動辨識以下中文欄位名稱：
- 銷貨日期
- 網路/實體
- 通路類型
- 品牌
- 代理經銷
- 數量
- 小計
- 合計
